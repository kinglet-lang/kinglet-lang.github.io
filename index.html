about

What ships today.

Kinglet is an experimental language. The toolchain you can install is a C++ bootstrap compiler (kinglet / klet) that lexes, parses, type-checks, lowers to KIR, and emits LLVM object code. There is no package manager, no stable stdlib guarantee, and no self-hosted compiler yet.

Language work pulls from WG21 papers that were deferred or dropped — concepts, tagged enums with match, uniform function call, T? nullables — but syntax and semantics are trimmed to fit a smaller whole. Details and examples are on the syntax page.

If you try it, expect compiler bugs and breaking changes between tags. Bug reports and PRs go to bootstrap.

install

Beyond the one-liner.

  • ▸ pin a prerelease The one-liner installs the latest stable release. Set this before running the script to force a specific tag, including -rc builds. KINGLET_VERSION=v0.1.0-rc.3
  • ▸ check the binary Install puts kinglet on your PATH. klet is the same executable. kinglet --version
  • ▸ manual install Download a release tarball and add the directory to PATH yourself. releases →

Kinglet is a small, natively-compiled language for people who build close to the machine — an ahead-of-time compiler, an explicit surface, and no runtime standing between your code and the hardware.

Get Kinglet Read the documentation →

Kinglet is early and experimental. The language and its compiler change between releases; the current toolchain is a reference implementation in C++ that emits native code through LLVM.

Why Kinglet

Compiled to the metal

Source is lowered ahead of time to native object code through an LLVM backend. There is no interpreter and no virtual machine to ship — the artifact is a binary you can run and profile like any other.

Familiar, not novel

The surface is deliberately close to the C family. Structs, enums, functions, and generics read the way a C++ or Rust programmer expects, so the language gets out of the way of the work.

Explicit over implicit

Control flow is visible, nullability lives in the type, and pattern matches are checked for exhaustiveness. The compiler prefers to reject an ambiguous program over guessing what you meant.

One toolchain

A single binary compiles, builds, and runs your project from a small manifest. Editor tooling and a matching color theme are developed alongside the compiler, in the open.

shapes.kl 
using io;

enum Shape {
  Circle(float),
  Rect(float, float),
}

float area(Shape s) {
  return s match {
    Shape::Circle(let r) => 3.14 * r * r,
    Shape::Rect(let w, let h) => w * h,
  };
}

int main() {
  Shape[] shapes = [Shape::Circle(2.0), Shape::Rect(3.0, 4.0)];
  for (int i = 0; i < shapes.len(); i = i + 1) {
    io::out.line("area = {}", area(shapes[i]));
  }
  return 0;
}

Community

Developed in the open.

Kinglet is a volunteer project. The language design, the compiler, and its tooling are discussed and built publicly on GitHub — proposals, bug reports, and patches are all welcome, and the direction of the language is set in the open.

Install

Try it in a minute.

One command installs the latest stable release and puts kinglet on your PATH. Then compile and run a file in a single step.

$ curl -fsSL https://kinglet-lang.org/install.sh | sh

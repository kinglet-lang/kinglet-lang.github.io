main.kl
module main;

}
pattern matching
pub fn evaluate(e: Expr) -> i32 {
    match e {
        Expr.Int(value)      -> value,
        Expr.Add(left, right) -> evaluate(left) + evaluate(right),
        Expr.Mul(left, right) -> evaluate(left) * evaluate(right),
    }
}

features

Designed for clarity, compiled for speed.

static types

Full type inference with explicit annotation support. Catch errors at compile time without ceremony.

pattern matching

Exhaustive match expressions on enums, structs, and primitives. No more cascading if-else.

modules

First-class module system with `pub` visibility and import resolution. Code at scale, cleanly.

fast compile

Built with a custom handwritten compiler. From source to binary in milliseconds.

LLVM backend

Generates optimized native code via LLVM. Same optimizer clang and rustc trust.

LSP ready

Go-to-definition, completions, diagnostics, formatting — first-class editor support out of the box.
